MIDAS DAQ package

You can find a pdf.tar.gz  documentation relative to the current Midas version under doc/midasdoc-pdf-x.y.z.tar.gz.
This documentation (midas.pdf) can be extracted with the following command:
 
> tar -zxvf midasdoc-pdf-x.y.z.tar.gz
Or you can access the online documentation either for the Swiss or Canadian site where ever is closer to you.
